## TL;DR - Copy & Paste After Installing Git

```powershell
cd "c:\Users\Fred\Desktop\Cybersecurity Projects\net-scan"
git init
git add .
git commit -m "Initial commit: NET_SCAN v1.0.0"
git remote add origin https://github.com/alfredsimeon/net_scan.git
git branch -M main
git push -u origin main
```

## Install Git First

**Option 1:** Download from https://git-scm.com/download/win

**Option 2:** Run in PowerShell (if Chocolatey installed):
```powershell
choco install git
```

**Option 3:** Windows Package Manager:
```powershell
winget install Git.Git
```

---

## What This Does

1. **cd** - Navigate to project folder
2. **git init** - Initialize repository
3. **git add .** - Add all files
4. **git commit** - Create first commit
5. **git remote add origin** - Connect to GitHub
6. **git branch -M main** - Rename branch to main
7. **git push -u origin main** - Upload to GitHub
